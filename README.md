# 🌌 DAARDOS | Senior Frontend Portfolio

Современное высокотехнологичное портфолио в стиле **Glassmorphism**, сфокусированное на визуальной чистоте и интерактивном пользовательском опыте.

## ✨ Особенности
- **Interactive Canvas Background:** Кастомная система частиц на JS, реагирующая на курсор и тач-события.
- **Glassmorphism UI:** Эстетичные карточки с эффектом размытия (backdrop-filter) и неоновыми акцентами.
- **Dynamic Content System:** Система вкладок (Profile/Focus/Goals) без перезагрузки страницы.
- **Responsive Architecture:** Полная оптимизация под мобильные устройства (снижение нагрузки на CPU/GPU).
- **Direct Mail Integration:** Интеграция с Formspree для прямой связи через Gmail.

## 🛠 Стек
- **Core:** HTML5 (Semantic), Modern CSS (Grid/Flexbox)
- **Animation:** HTML5 Canvas API, CSS Keyframes
- **Backend:** Formspree API (Contact Form)
- **Icons:** FontAwesome 6
